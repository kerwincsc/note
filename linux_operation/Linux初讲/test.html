第一天习题

  1. Unix 和 Linux之间有什么关系？
    UNIX
    属于分时操作系统,诞生于Bell Labs1,由 Ken Thompson 在MULTICS计划被迫停止后于19702年开发出来;
    Linux
    实际是linux只是一个由 Linus Torvalds 在Minix的影响下写出的操作系统内核,它与GNU工具共同组成了我们常说的 Linux操作系统 ;
    Minix
    是一个不使用任何AT&T的源代码前提下，与UNIX兼容的操作系统,由Andrew S. Tanenbaum(塔能鲍姆教授)为教学目的自行开发的;属于类UNIX系统;

有的人说Linux是从UNIX衍生过来的,这么说并不准确.Linux从Minix衍生而来,而Minix是不使用UNIX源码的类UNIX系统,上面有提到.Linux继承了UNIX以网络为核心的设计思想;

  1. BSD是什么？ 我们通常说的FreeBSD、NetBSD和BSD又有什么关系呢？ BSD3是Unix的衍生系统,在1977至1995年间由加州大学伯克利分校开发和发布的;

    FreeBSD是经过BSD,386BSD和4.4BSD发展而来;

    NetBSD是从4.4BSD Lite衍生而来;

    总的来说,这两款BSD是基于BSD的类Unix操作系统;

  2. Unix许可证禁止在大学使用Unix源码是哪一年，随后就出现了两个重要的Unix版本线，分别是什么呢？ 20世纪70年代，AT&T公司开始注意到Unix所带来的商业价值。公司的律师开始寻找一些手段来保护Unix，并让其成为一种商业机密。从 1979年Unix的版本V7 开始，Unix的许可证开始禁止大学使用Unix的源码，包括在授课中学习;

    到了1980年，有两个最主要的Unix的版本线，一个是 Berkeley的BSD UNIX ，另一个是 AT&T的Unix ，在这个时候，很显然，竞争最终引发了Unix的战争。在这场战争中，好的是，软件开发人员还是能够得到Unix的源码并对其按照自己的需要和兴致进行裁剪。而不好的是，Unix开始一发不可收拾地开发不停地出现各种各样的变种;

  3. AIX, Solaris, HP-UX, IRIX这些Unix都是哪些公司开发的呢？它们是基于哪个分支的UNIX衍生（第3题中的两个版本线）？
    • AIX是由IBM开发的一套 UNIX 操作系统; 基于 AT&T Unix System V
    • Solaris是Sun公司研制的类Unix操作系统; 早期基于 BSD Unix ,因为其创始人Bill Joy(比尔乔伊)来自加州大学伯克利分校,后来其在接口上向System V靠拢;
    • HP-UX是惠普公司以SystemV为基础研发的类Unix系统; 基于 System V
    • IRIX是由硅谷图形公司(Silicon GraphicsInc,简称SGI)以System V与BSD延伸程序为基础发展而成的UNIX操作系统;
  4. 请把GNU 和 GPL 这两个概念搞明白，请用最简短的话描述一下。
    GNU
    由Richard Stallman发起,旨在创建一套完全自由的操作系统的计划;
    GPL
    通用公共许可证,旨在保证你共享和修改软件的自由;
  5. GPLv2 和 GPLv1比有哪些变化呢？ GPLv2与GPLv1最大的不同是, v2中增加了"自由还是死亡"("Liberty or Death")这章条款.这章中申明道，如果哪个人在发布源于GPL的软件的时候，同时添加强制的条款，以在一定程度上保障和尊重其它一些人的自由和权益（也就是说在一些国家里，人们只能以二进制代码的形式发布软件，以保护开发软件者的版权），那么他将根本无权发布该软件。

  3. 请查资料了解密钥对的工作原理。 密钥对中,公钥放在远程机上,负责对消息进行加密,而与公钥相匹配的私钥在本地用来解密消息;

    当使用密钥对登录系统时,远程机的sshd会生成一个随机数,并用拷贝过来的公钥对这个随机数进行加密,然后,sshd把加了密的随机数发回给本地机上的ssh.本地ssh使用给定的私钥对这个加了密的随机数进行解密后,再发回给远程机的sshd.得到正确匹配后,表明本地机持有与其相对应的私钥,就会允许登录;
