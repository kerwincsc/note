* shell基础
** if逻辑判断
   shell中的逻辑表达式与一般逻辑表达式对照
   | 一般逻辑运算符 | shell的中逻辑运算符 |
   |----------------+-------------------|
   | <c>            | <c>               |
   | >              | -gt               |
   | <              | -lt               |
   | 等于==         | -eq               |
   | >=             | -ge               |
   | <=             | -le               |
   | !=             | -ne               |
